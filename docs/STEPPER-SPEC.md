# Спека степпера glebkudr.com (снято с живого сайта, секция «Мой метод»)

Воспроизвести 1:1 для наших степперов (#how «Как попасть» и «Неделя»).

## Desktop
- **Контейнер:** `display: grid`, равные колонки (`repeat(N, 1fr)`), `gap: 0`. Шаги БЕЗ рамок.
- **Шаг:** `background: transparent`, без `border`/`box-shadow`/`border-radius`,
  `padding: 0 20px 0 0` (правый отступ под стрелку), `display: block`.
  Порядок элементов сверху вниз:
  - **номер:** `<span>` моно, `font-size: 15px`, `font-weight: 800`, цвет синий `#0A67FF`
  - **заголовок:** `<h3>` Inter, `16px`, `font-weight: 700`, navy `#06111F`, `margin-bottom: 16px`
  - **иконка:** `svg` `46×46px`, синяя (`rgb(22,114,255)`), `margin-bottom: 14px`
  - **описание:** `12px`, navy, `line-height: 1.28`
- **Стрелка между шагами:** символ `→`, `font-size: 38px` (КРУПНАЯ), цвет navy `#06111F`,
  `position: absolute` в правом промежутке шага, по центру по вертикали.
- Цветовое кодирование шагов: blue / navy / green / mint (номер/иконка в цвет).

## Mobile (≤760px)
- `grid-template-columns: 1fr`, `gap: 16px`.
- Шаг ГОРИЗОНТАЛЬНЫЙ: иконка слева, справа — номер + заголовок + описание.
- Стрелки между шагами СКРЫТЫ.

## Применить
- #how «Как попасть» (3 шага: Заявка / Интервью / Первая волна) — нужны 3 svg-иконки
  (документ-заявка / диалог-интервью / люди-когорта).
- «Неделя» (4 шага пн/ср/пт/···) — привести к тому же открытому виду; вместо номера —
  день (пн/ср/пт) тем же моно-стилем, иконки по смыслу.
- Стрелки крупные navy в промежутках (desktop), скрыты на mobile.
- УБРАТЬ «путь резидента» (чек-лист) из #how совсем.
