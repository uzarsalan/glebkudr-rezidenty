# Фикс вёрстки (дефекты, найденные conductor через поэкранный скан)

Контент и токены НЕ трогаем — только геометрия/отступы/пропорции.

1. **Вертикальные промежутки между секциями — ГЛАВНОЕ.** Сейчас между почти
   всеми блоками 120–180px пустоты, лендинг «разваливается». Сократить
   section padding/margin (ориентир: вдвое меньше текущего, плотнее как у glebkudr).

2. **Блок 3 «ИИ-самооценка» (полная, тёмный терминал #assess).** Терминал
   неоправданно высокий: поле ввода и кнопка тонут в пустоте сверху/снизу.
   Ужать высоту терминала под контент (убрать лишние min-height/padding).

3. **HERO — переосмыслить компоновку (не просто выровнять).**
   Целевая компоновка (2 колонки, выровнены по высоте, без повисшей пустоты):
   - ЛЕВО (~42%): kicker → H1 (крупный, плотный) → подзаголовок → кнопки в один
     ряд (на узком перенос) → note. Прижата к верху.
   - ПРАВО bento (~58%): сверху 2 КОМПАКТНЫЕ карточки в ряд — spec-терминал +
     чек-лист «рабочий процесс»; под ними agent_trace · радар во всю ширину bento;
     внизу console · самооценка во всю ширину (главный интерактив, акцент).
   - Высота правой ≈ левой, низ ровный, убрать разрыв hero → «Проблема».
   - Карточку-«документ» с чертежом УБРАТЬ из hero и перенести в блок «Диплом»
     (#diploma) — там она по смыслу на месте, hero не перегружен.

4. **Карточки тарифов не выровнены по низу.** У «Наблюдателя» (меньше пунктов)
   кнопка висит выше, под ней пустота. Прижать кнопки к низу (flex + margin-top:auto),
   обе карточки равной высоты.

5. **Сноски наезжают на край карточек.** «Это ещё не интервью…» (под терминалом
   самооценки) и «Ответим за два дня…» (под формой apply) вылезают левее блока —
   выровнять по контейнеру карточки.

6. Мелочь: в цитате Глеба (#gleb) убрать тире («Курсы я уже прошёл — они кончаются»
   → «…прошёл, они кончаются»); в блоке «Почему работает» цветные квадраты-иконки
   пустые — либо вложить простой значок/глиф, либо убрать квадраты.

Проверка: пересними весь лендинг ОДНОЙ короткой серией (browse теряет страницу в
длинных циклах — снимай пачкой по 4-5 кадров за вызов). НЕ деплой.
