## Радиусы и формы
- Основной радиус карточек/кнопок: **8px** (доминирует на сайте).
- Пилюли/теги/бейджи: **999px** (полное скругление).
- Мелкие элементы: 6–7px.

## Сетка и отступы
- Базовый шаг spacing: `0.25rem` (Tailwind).
- Контейнер контента: max-width **~72rem** (1152px), узкий текстовый блок ~42rem.
- Секции: крупные вертикальные паддинги (воздух), `clamp()` для адаптива.

## Кнопки
- Primary: фон `--accent` (#fe6e00), текст белый, radius 8px, hover → `--accent-700`.
- Secondary/ghost: прозрачный фон, рамка `--ink` с низкой непрозрачностью, текст navy.

## Фирменные приёмы (перенесены в лендинг)
1. **Нумерация секций** моноширинным: `01 — проблема`, `02 — …`.
2. **Стрелки `→`** для вывода/итога мысли.
3. **Код-бейджи**: `пн`, `ср`, `пт`, теги тарифов, метка радара.
4. Тёмная секция-акцент (hero/радар) на `--dark` с инверсией текста.
5. Тонкие разделители `1px` на `#06111f14`.

## Доступность
- Контраст `#253347` на `#f7f2e8` ≈ 9:1 (AA+).
- Акцент `#fe6e00` используется как фон только с белым текстом для CTA;
  как текст — на тёмном/белом, не на бумаге для длинных строк.
