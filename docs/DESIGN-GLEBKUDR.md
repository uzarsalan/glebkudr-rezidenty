## Метафора (задумка)
Рабочий стол / лабораторный журнал инженера. Не продающая простыня, а «верстак»:
светлая бумага в фоне, карточки-листы, скрепка, чертёж, тёмные терминалы с живым кодом,
чек-лист «рабочий процесс», timeline метода. Доверие через демонстрацию реальной работы:
скевоморфизм бумаги (ремесло) + терминалы (технологичность).

## Токены (точные, computed)
- **Шрифты:** `Inter` (заголовки и текст) + `ui-monospace, SFMono-Regular, Menlo, Consolas, monospace` (код, подписи, номера). НЕ Unbounded, НЕ Commissioner.
- **Фон страницы:** `rgb(251, 247, 238)` (тёплый кремовый, бумага). Доминирует светлый.
- **Радиальные блики в фоне:** `radial-gradient(circle at 77% 8%, rgba(7,103,255,0.055), transparent 24%)` — лёгкая синяя подсветка угла. Можно добавить 1-2 таких блика.
- **Основной текст/чернила:** navy-чёрный `rgb(6, 17, 31)`.
- **Акцент:** синий `#0767FF` (rgb(7,103,255)) — в моно-подписях, номерах, ссылках. Оранжевого НЕТ.
- **Доп. акценты иконок** (на тёмных карточках, разноцветные): жёлтый, фиолетовый, cyan, зелёный — каждая услуга своим цветом.

### Тёмные карточки (терминалы, услуги)
- bg: `rgba(6,15,22,0.94)` / `rgba(5,14,21,0.96)` (почти чёрный navy)
- radius: `8px`
- shadow: `rgba(0,0,0,0.32) 0px 24px 42px 0px` (глубокая мягкая)
- внутри: моноширинный цветной «синтаксис» — синие/розовые/зелёные/cyan токены

### Светлые карточки (бумага-листы)
- bg: `rgb(244,239,229)` … `rgba(251,247,238,0.98)`
- radius: `8–10px`
- border: `1px solid rgba(6,17,31,0.16)` (тонкая тёмная обводка)
- shadow: `rgba(6,17,31,0.18) 0px 18px 34px 0px` (приподнятый лист)
- скевоморфизм: скрепка-уголок, у одной — чертёж/план («документ»)

## Структура (порядок секций как у них)
1. **Header** — лого слева (квадрат-марка + «ИИ-лаборатория…»), справа nav + кнопки.
2. **Hero = bento-дашборд** (НЕ письмо!):
   - слева: H1 крупно + подзаголовок + кнопки (1 тёмная primary + 2 светлые-обводка);
   - справа: сетка карточек — терминал «// спецификация» (цветной код), терминал «AGENT TRACE» (лог), «REVIEW / DIFF» (красно-зелёный дифф), карточка «документ» с чертежом + скрепка, белая карточка «Рабочий процесс» с чек-листом и зелёной ✓.
3. **Карточка автора** — фото Глеба + имя/регалии + цитата с большими кавычками + блок «Контакт».
4. **Секция тезисов** («// моя позиция») — 3 светлые нумерованные карточки 01/02/03.
5. **Услуги** («// с чем я помогаю») — тёмные карточки с круглыми разноцветными иконками.
6. **Timeline «Мой метод»** — 01→02→03→04→05 со стрелками-коннекторами и иконками, номера синие моно.
7. **Баннер-полоса** — крупная фраза на всю ширину.
8. **Варианты платы / тарифы** — карточки (у них тёмные + фото).
9. **Footer** — тёмный navy с моно-колонками.

### Секционные подписи
Моно капсом с `//` и цветом-акцентом: `// моя позиция`, `// с чем я помогаю`, `// мой метод`.
(У нас сейчас «01 — проблема» — заменить на этот формат.)

## Как ложится НАШ контент на их скелет
- Hero-bento: H1 (наш) + кнопка «Пройти интервью». В правую сетку карточек вмонтировать
  НАШИ живые ИИ-механики как терминалы: «Радар недели» = терминал AGENT TRACE (живой!),
  «ИИ-самооценка» = терминал-консоль с вводом. Это и красиво, и по делу.
- Письмо Глеба → в карточку автора (секция 3), сокращённо, как цитата/манифест.
- Блоки проблема/что это/почему/диплом/рекомендации → светлые нумерованные bento-карточки.
- «Неделя» и «Как попасть» → timeline со стрелками (как «Мой метод»).
- Тарифы → карточки (резидент светлая/выделенная, наблюдатель тёмная или наоборот).
- Тёмные полноширинные секции убрать; тёмное — только в карточках-акцентах.
